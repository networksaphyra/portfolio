My Recent Projects


Here are some of the recent projects I've worked on.

Sky Scope

An automated telescope that can locate and track celestial objects, eliminating manual adjustments. It integrates with a celestial database containing coordinates for stars, planets and other space objects. Users select viewing targets through a web app which calculates their real-time positions based on the user's location.

The Judge

An intelligent grading assistant for computer science projects. The Judge uses natural language processing models like GPT to analyze and provide feedback on submitted code, documentation and requirements across different programming languages. Students and instructors submit projects through a web interface built with HTML, CSS, and Javascript.

Project Astraeus Website

A comprehensive website for the non-profit Project Astraeus which runs STEM education initiatives. The site design emphasizes clear navigation to information on their future goals, workshops and resources for getting youth engaged with engineering.

Smart Sprout

An automated plant watering system that ensures your plants never go thirsty. Smart Sprout uses calibrated moisture sensors inserted into planters to closely monitor soil conditions. It then automatically dispenses precise levels of water to each plant only when needed. Smart Sprout's integrated pump and tubing efficiently distributes water while an LED display is used to keep the user informed about the moisture.

Sleepy Joe

A web-based conversational app designed to help users fall asleep more easily. Sleepy Joe lets you interact with an AI model that can write and narrate custom bedtime stories tailored to your preferences. You can also ask it to display calming images of itself sleeping or play soothing audio tracks and ambient sleepy sounds.

Portfolio Site

A clean and modern website showcasing my projects, skills, and experience as a developer.

Chess

A classic chess game you can play locally against friends or a computer. Built using Pygame, the clean graphical interface allows you to click and drag pieces across the virtual board. Working on adding features that include the ability to undo moves, allow playing across networks, and save/load game states.

Alice in Quizzland

A web app for creating and taking quizzes on any topic. Build custom quizzes by adding multiple choice, true/false, short answer questions and more. Take quizzes and get comprehensive results showing your performance. Built using React, Node.js and MongoDB.

Sketcher

A browser-based pixel art drawing tool. Create digital pixel artworks directly in your web browser using a simple and intuitive drawing interface. Sketcher provides a variety of tools like pencils, buckets, color pickers to help you construct detailed pixel art images.
